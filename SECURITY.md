# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately.

Preferred: open a GitHub Security Advisory for this repository.

If you cannot use Security Advisories, create a GitHub issue with minimal details and ask for a private channel to share a full report.

When reporting, include:
- A description of the vulnerability and impact
- Steps to reproduce (proof-of-concept if available)
- Affected versions/commits and any relevant configuration

## Supported Versions

Only the latest `main` branch is supported with security updates.
